# hugo-notebox
Note Boxes for Hugo using TailwindCSS

You can use the format:

`{{< notebox warning >}}
This is a warning!
{{< /notebox >}}

{{< notebox note >}}
A note for you.
{{< /notebox >}}

{{< notebox tip >}}
Just a tip.
{{< /notebox >}}

{{< notebox info >}}
Some info for you
{{< /notebox >}}  

{{< notebox trivia >}}
Funny note...
{{< /notebox >}}`
